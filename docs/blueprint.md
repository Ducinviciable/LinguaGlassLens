# **App Name**: LinguaLens

## Core Features:

- Screen Capture: Continuously capture a user-defined region of the primary screen at a specified frame rate.
- OCR Extraction: Extract text from captured images using OCR technology (Tesseract).
- Real-time Translation: Translate extracted text in real-time using an AI translation tool and automatically detect source language.
- Secondary Monitor Display: Display the translated text on a secondary monitor in a customizable overlay window.
- User Interface: Provide a minimal GUI for controlling translation, language selection, region selection, and refresh rate.
- Error Handling: Gracefully handle common errors such as no text detected or API rate limits.
- Settings: Option to change font size, background color and opacity of the overlay.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082), creating a focused and sophisticated ambiance, avoiding any distractions.
- Background color: Very dark indigo (#1A0033), providing a contrasting backdrop that allows the text to stand out clearly while maintaining consistency.
- Accent color: Soft violet (#8A2BE2), used sparingly to highlight key controls and interactive elements.
- Font for all text: 'Inter', a sans-serif with a modern, neutral appearance suitable for readability.
- Simple, geometric icons for controls. Preferably monochrome, following the accent color.
- Clean, uncluttered layout focused on essential controls and clear translation display.
- Subtle fade-in/out animations for text updates on the secondary monitor to reduce visual distraction.
